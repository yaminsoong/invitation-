# Undangan Pernikahan Ahmad & Anisah

Website undangan pernikahan sederhana dengan tema adat Jawa, menggunakan motif parang dan wayang kulit. Dibuat dengan HTML, CSS, dan JavaScript ringan untuk kemudahan hosting.

## Fitur
- Desain responsif dengan elemen SVG
- Form kartu ucapan yang redirect ke WhatsApp
- Tema adat Jawa dengan warna dan motif tradisional

## Cara Menggunakan
1. Upload file `index.html`, `style.css`, dan `script.js` ke hosting static Anda (misalnya GitHub Pages, Netlify, atau hosting web biasa).
2. Buka URL website untuk melihat undangan.

- **Pengantin Pria:** Ahmad Askhabul Yamin (anak Bapak M. Hasan & Ibu Sofiah)
- **Pengantin Wanita:** Anisah Permata Sari (anak Bapak Siman & Ibu Titin)
- **Tanggal:** 29 Desember 2025
- **Waktu Akad:** 09:00 WIB - Selesai
- **Lokasi:** Kediaman Mempelai Wanita, Talesan Jajar, Purwantoro, Wonogiri

Untuk kartu ucapan, pengunjung dapat mengisi form dan pesan akan dikirim via WhatsApp.
